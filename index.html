How it works

Darken the MacBook screen only when it is duplicating another display.

DisplayHelperPro is made for desks, demos, and presentation setups where the built-in screen keeps glowing even though the external display is the one you are actually using. It can also toggle built-in-to-external mirroring for you and reuse the last external display you selected.

Mirror MacBook screen duplicates the external display built-in screen goes dark
Extend MacBook screen is part of your workspace screen stays available
Interactive preview

Smart enough to know the difference.

Plugging in a monitor should not mean losing your MacBook screen. The app keeps extended workspaces available, watches for display changes, and reapplies blackout after wake only when the built-in panel is mirroring another display.

Display adjustment

Pro uses deeper display-control paths than the App Store edition.

The Pro build is distributed directly so it can use macOS display brightness APIs that are not suitable for App Store submission. For external displays, it tries DDC/CI hardware brightness and contrast controls when the monitor supports them, then falls back to software gamma adjustment when needed.

Built-in blackout Darkens the MacBook panel when CoreGraphics reports it is mirroring another display.
External controls Uses monitor hardware controls for brightness and contrast where available.
Software fallback Uses display transfer tables when hardware controls are unavailable or ineffective.

Privacy & safety

No data collection. No analytics. No tracking.

DisplayHelperPro's display-management features run locally on your Mac. The app does not collect, store, sell, share, or transmit personal information through display management. Update checks contact the Sparkle feed, license actions contact Lemon Squeezy, and feedback items open an email draft that you review before sending.

Trial & license

Full Pro controls during the trial, then a local license check.

The latest Pro source includes a 14-day full-feature trial, Lemon Squeezy license activation, validation, and deactivation, plus a 7-day offline grace period after a successful validation.

License data stays small and local. The license key and local license instance are stored in Keychain. Trial and validation timestamps are stored locally so Pro can work through ordinary offline use.
