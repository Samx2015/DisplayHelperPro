How Pro works

Darken the MacBook screen only when it is mirroring.

  • Detects mirroring checks whether the built-in panel is duplicating an external screen
  • Leaves extended mode alone keeps the MacBook screen usable when it is part of your desktop
  • Recovers after wake rechecks layout and reapplies blackout when mirroring returns
Mirror built-in panel goes dark while the external display stays active
Extend both screens remain available as one workspace
Away dim everything until the next key press
Menu controls

One menu covers the daily controls.

  • Screen actions blackout, mirroring, dimming, sleep, theme, shortcuts, brightness, and contrast
  • Network row current and alternative BSSID, colored for quick recognition, with a threshold slider
  • Housekeeping login startup, updates, license state, about, privacy, and feedback
Why Pro is direct-download

The direct-download build can use APIs App Store apps cannot.

  • Display access uses macOS brightness APIs plus external-monitor DDC/CI when available
  • Cautious roaming uses CoreWLAN status, BSSID-aware candidates, throttled scans, and a threshold
  • Independent updates uses Sparkle updates and a local Keychain-backed trial/license state
Display backend private brightness APIs, DDC/CI where available, and gamma fallback
Distribution Developer ID signed, Apple-notarized, and updated through Sparkle
Network awareness current-strength polling and candidate tracking by router address
License 14-day full-feature trial with local Keychain-backed license state

Privacy

No account. No analytics. Review before sending.

  • Local by default no analytics, account, background telemetry, or personal data sale
  • No password access shows network status without reading or storing Wi-Fi passwords
  • Reviewable contact feedback opens an email draft; updates and licenses contact their services only when needed
