How Pro works

Darken the MacBook screen only when it is mirroring.

DisplayHelperPro watches your display setup, keeps extended workspaces available, and reapplies blackout after wake or display changes only when the built-in panel is duplicating another display.

Mirror built-in screen goes dark while the external display stays active
Extend the MacBook screen stays available as part of your workspace
Away dim displays until the next key press, then restore them

Why Pro is direct-download

Deeper display controls, Sparkle updates, and a Pro license.

The direct-download build can use macOS display brightness APIs that are not suitable for App Store submission. External displays use DDC/CI hardware controls when supported, with software gamma adjustment as a fallback.

Display backend private brightness APIs, DDC/CI where available, and gamma fallback
Distribution Developer ID signed, notarized, and updated through Sparkle
License 14-day full-feature trial with local Keychain-backed license state

Privacy

No account. No analytics. Review before sending.

Display management runs locally. The app does not collect, sell, share, or transmit personal information through normal display controls. Update checks contact Sparkle, license actions contact Lemon Squeezy, and feedback opens a reviewable email draft.
